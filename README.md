# web-ssh
Web ssh client

## 介绍
一个web ssh客户端，可以通过界面上连接访问远程服务器。

![新建连接](img/create-connect.png)

## 技术栈
### Backend
- SpringBoot
- WebSocket
- [SSHJ](https://github.com/hierynomus/sshj)
- [ExpectIt](https://github.com/agavrilov76/ExpectIt)

### Frontend
- [Vue3](https://vuejs.org/)
- [Xterm](https://github.com/xtermjs/xterm.js)
